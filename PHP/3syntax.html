Syntax

PHP can placed anywhere in the .php File.

<?php and ends with ?> 


            <?php
                //PHP code goes here
            >

Default file Extension for PHP is ".php".

A PHP file normally contains HTML tags, and some PHP scripting code.


            <!DOCTYPE html>
            <html>
            <body>

            <h1>My first PHP page<h1>

            <?php
            echo "Hello World!";
            ?>

            </body>
            </html>

Output:

My first PHP page

Hello World!

PHP Case Sensitivity

In PHP, Keywords-if,else,while,echo,etc., classes,Functions, and user-defined functions are not case-sensitive.

Example



            <!DOCTYPE html>
            <html>
            <body>

            <h1>My first PHP page</h1>

            <?php
            echo "Hello World!<br>";
            ECHO "Hello World!<br>";
            EcHo "Hello World!<br>";
            ?>

            </body>
            </html>

Output:

My first PHP page

Hello World!

Hello World!

Hello World!
