Installing PHP

Install a Webserver.

Xampp server is used for all the OS.

X-Multiplatform, A-Apache, M-MariaDB, P-Perl, P-PHP

Mamp server is used for MAC OS.

M-Mac,A-Apache,M-Mysql,P-PHP.(command line for other languages like- Python,Ruby and Pearl.)

Wamp server is used for Windows OS.

W-Windows,A-Apache,M-Mysql,P-PHP.

LAMP server is used for Linux OS.

L-Linux,A-Apache,M-Mysql,P-PHP,Perl and Python.

PHP.

PHP 8.4.2 is the Latest Version.

Use php.net Website to Download the Latest Version of the PHP.

echo phpversion(); To View the PHP Version by the PHP Code.
